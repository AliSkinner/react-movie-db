# The Movie DB

Search for movies and checkout their details, with support from The Movie DB.

## To run locally
```
git clone git@github.com:AliSkinner/react-movie-db.git
cd react-movie-db
yarn
yarn start
```

## To test
```
git clone git@github.com:AliSkinner/react-movie-db.git
cd DAZN
yarn test
```
